# Student Database Management System

A professional **MySQL Database Management System** project developed to demonstrate SQL concepts ranging from basic database creation to advanced database objects such as Views.

This project is designed for learning, interview preparation, and showcasing SQL skills on GitHub.

---

# 📌 Features

* Database Creation
* Table Creation
* Sample Data Insertion
* CRUD Operations
* Basic SQL Queries
* JOIN Operations
* Aggregate Functions
* GROUP BY & HAVING
* Subqueries
* Transactions (COMMIT & ROLLBACK)
* Indexes
* Views

| --------------------------- | ---------------------------------------------------------- |

| 11_indexes.sql              | Index creation and query optimization                      |
| 12_views.sql                | Simple and complex SQL Views                               |
